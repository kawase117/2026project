## 💡 訓練期間による効果の違い

### 重要な発見：「グループ組成の変化」

各訓練期間で、「高勝率グループ」は以下のように定義されます：
```
高勝率グループ = 訓練期間内での勝率が「中央値以上」の機種群
```

訓練期間が短いほど、各機種のサンプル数が少なく、勝率計算が不安定になります。
その結果、中央値での分割時に**グループに属する機種セットが変わります**。

#### 例：D4 機種名
```
6月訓練:  中央値が機種{A,B,C}を分ける
         → 高勝率グループ = {A,B,C の一部}
         → その平均: 44.4%

3月訓練:  中央値が機種{A,B,C}を異なる位置で分ける
         → 高勝率グループ = {A,B,C の別の組合}
         → その平均: 43.8%（機種セットが異なる！）
```

### パターン1: 信号が弱化（D11, D20）
```
D11: 6月 +10.8% → 3月 +12.2% → 1月 +4.4%
D20: 6月 +6.2%  → 3月 +6.4%  → 1月 +2.0%
```
**解釈**: 1ヶ月訓練は信号が著しく減弱。3ヶ月が最適。

### パターン2: グループ組成の影響（D4）
```
D4: 6月 -0.6% → 3月 -1.2% → 1月 +2.3%
```
**解釈**: 数値の変化は「訓練期間による信号変化」ではなく「グループ組成の変化」を反映

## 🔴 1ヶ月訓練での「勝敗再現性」（機械番号特別処理）

### 勝敗再現性の定義

訓練期間（3月）で「勝った台」と「負けた台」に分類し、テスト期間（4月）でそれが再現するかを判定。

**例：D1機械番号**
```
訓練期間勝利台：59台が勝った
  → テスト期間での勝率: 39.9%（148台中59台が勝利）

訓練期間敗北台：74台が負けた
  → テスト期間での勝率: 36.6%（202台中74台が勝利）

判定: 訓練期間勝者が優位（39.9% > 36.6%） ✅
```

### 勝敗再現性の傾向

#### 訓練期間勝者が優位の日付
- **D1, D3, D4, D5, D9, D14, D16, D17, D19, D20** （10/20日）
- 勝敗の再現性がある程度存在

#### 訓練期間敗者が優位の日付
- **D2, D6, D7, D8, D10, D11, D12, D13, D15, D18** （10/20日）
- 逆説的：敗北した台が次月で勝つ傾向

**⚠️ 重要**: 1ヶ月の機械番号は、50/50のコイン投げに近い。単純には予測できない。

## 📈 訓練期間別の推奨活用

### 6ヶ月訓練の使い方
✅ **最も信頼できる**
- 属性の「本質的な強さ」を反映
- 市場の季節変動を平準化
- マージン ±5% 以上なら強い信号

使用目的：
- 長期的なランキング確定
- 機種評判の確定（機種名属性）

### 3ヶ月訓練の使い方
✅ **実用的で信頼できる**
- 直近の市場動向を反映
- 相対値が6月とほぼ一致するなら「確度が高い」
- 6月と異なるなら「市場環境が変わった」

使用目的：
- 実運用での日次判断
- 月次の更新予測

### 1ヶ月訓練の使い方
⚠️ **特殊な用途に限定**
- 機械番号：勝敗再現性に注目（確率的には五分五分に近い）
- 機種名：信号が弱まるため、3ヶ月と明らかに異なる場合のみ信頼
- 台末尾：ほぼ無視

使用目的：
- 「今月急に変わった」という検知用
- 短期トレンド把握

## 🎓 実装上の推奨

### 段階的な意思決定フロー

```
1️⃣ 6月訓練のシグナル確認
   → マージン < ±3% ？ 弱い信号 → 採用しない
   → マージン > ±5% ？ 強い信号 → 次へ

2️⃣ 3月訓練で確認
   → 6月と同じ方向 ？ → 確度が高い ✅
   → 6月と逆方向 ？ → 市場環境が変わった ⚠️

3️⃣ 1月訓練で異常検知
   → 3月と大幅に異なる ？ → 月内トレンド変化 📊
```

### 属性別の推奨訓練期間

| 属性 | 推奨訓練期間 | 理由 |
|------|-----------|------|
| **機種名** | **6月** | 最も安定した予測力 |
| **機械番号** | **3月** | 相対値が安定。6月との一致で確度判定 |
| **台末尾** | **非推奨** | 市場環境に埋もれやすい |

## ✅ 結論

### 訓練期間の長さ = 信号の信頼度

1. **6ヶ月** → 最も信頼できるが、市場が変わった場合を検知できない
2. **3ヶ月** → 実運用での最適バランス
3. **1ヶ月** → 異常検知用（確率的な信号は弱い）

### 最適な運用モデル

**メイン予測**: 3ヶ月訓練の相対値（ただし6月訓練と一致することで確度向上）  
**確度判定**: 6月訓練との比較確認  
**異常検知**: 1ヶ月訓練での大幅な乖離
