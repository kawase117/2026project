 • 機械番号分割（DD別）：50.0% → 業界平均比 +15%

実装優先順位
================================================================================

【フェーズ1（即導入）】実装難度：★☆☆ / 効果：★★★

機械番号による「勝ち台」の重点管理

実装内容：
 1. 前月の全台差枚をランキング
 2. 上位36%をTOP台として特定
 3. 当月の各DDで同じ台を優先配置
 4. 50%の確率で「継続好調」を実現

具体例：
 • 前月全台100台中、差枚TOP36台 → 当月も各DDでこれらの台を配置
 • D1～D20各日で同じ台を追跡配置することで、18台程度の継続好調を保証
 • 月末に「勝ち台リスト」を更新（新台導入時は除外）

================================================================================

【フェーズ2（高優先度）】実装難度：★★☆ / 効果：★★★★

機種×曜日の相互作用を活用（71.4%精度！）

実装内容：
 1. 前月の曜日別×機種別の差枚マトリクスを作成
 2. 各曜日の上位3機種を特定
 3. 当月の機種交換・配置に反映
   ├─ 土曜日に交換予定 → 前月土曜TOP3機種を優先配置
   ├─ 金曜日新台導入 → 前月金曜好調機種と同機種を選択
   └─ 各曜日で「機種×差枚」の傾向を学習・予測

サンプル実装：
 前月の曜日別TOP機種（架空例）
  月曜：北斗シリーズ、バジリスク、ゴッド
  火曜：北斗シリーズ、バジリスク、リノ
  水曜：グレートキング、バジリスク、ゴッド
  木曜：北斗シリーズ、グレートキング、ファラオ
  金曜：バジリスク、ゴッド、リノ
  土曜：ファラオ、グレートキング、北斗シリーズ ← 高額消費層向け
  日曜：北斗シリーズ、バジリスク、グレートキング ← ファミリー向け

 → 当月の同曜日で上記機種をまた配置する

効果：71.4% の確率で「上位グループ」が勝利
    = 7日中5日は最高の差枚実績が見込める

【フェーズ3（補助調整）】実装難度：★★★ / 効果：★★

機種の世代別分析（新規 vs 既知）

新台導入後の変動パターンを学習：
 • 導入1-3か月：機械番号と機種特性が未分離 → 機種分割を優先
 • 導入4か月以上：機械番号の個体差が出現 → 機械番号優先に切り替え

季節トレンドの考慮：
 • GW・盆・年末：客層が変わり、機種×曜日パターンが逆転する可能性
 • → 当月が「特殊期間」なら、通常ルールを50%程度減衰させる

DD別・曜日別の詳細戦略
================================================================================

【月初：D1～D5】危険な「逆転期」
 ├─ D1初日：下位G 50% ← 低差枚台スポット導入で逆転狙い
 ├─ D2-D4：上位G 45-50% ← 勝ち台（前月TOP36）を集約配置
 └─ D5：下位G 50% ← バランス配置で対策

【月中：D6～D14】「安定優位期」
 ├─ D6-D10：上位G 50% ← 機械番号による勝ち台重点
 ├─ D11-D12：上位G 45-50%
 ├─ D13：上位G 100% ★★★ 「勝率保証日」 ← 新台トライアル最適
 └─ D14：上位G 67% ← 積極的な新投入期

【月末：D15～D20】「乱高下期」
 ├─ D15-D17：上位G 0-33% ← 危険期間（低差枚台優位）
 ├─ D18-D19：上位G 33-50% ← D1-5と同じ逆転パターン
 └─ D20：上位G 67% ← 回復傾向（勝ち台再配置）

【曜日別スペシャル】
 ├─ 月曜：平日（シニア客） → 機種×曜日が重要
 ├─ 火-木：通常営業 → 機械番号優先
 ├─ 金曜：週末前（若年化） → 機種特性が強まる
 ├─ 土曜：ピーク（全層） → 機種×曜日が最大に表出（71.4%）
 └─ 日曜：ファミリー（子連れ） → 機種×曜日で大型機が有利

KPI測定とチューニング
================================================================================

月間目標値：
 ✓ 機械番号TOP36台の当月中での「上位50%到達率」：100%
   （理論値50% × 20日 = 10日以上が目標）

 ✓ 曜日×機種パターンの「上位グループ達成率」：70%以上
   （理論値71.4% ≈ 7日中5日以上）

 ✓ 月間差枚伸び率：前年同月比 +5%～+10%

 ✓ 配置台当たりの差枚標準偏差：-20%削減

失敗シグナルと対策：
 • 50%精度が2か月連続で未達 → 分割比率を45/10/45に変更テスト
 • 71.4%が達成不可 → 機種サイクルの変化が大きい（新台多い）
   → 機種世代別（新規 vs 既知）での再分析を開始
 • 月間差枚が前年同月比で-5%以下 → ホール特性変化の可能性
   → マクロ環境要因（客数減、競合出店等）を調査
