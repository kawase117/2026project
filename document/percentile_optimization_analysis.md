## 分割比率タイプ別の特性

### 2値分割 (50/0/50) - 上位と下位のみ

```
50% (上位G) | 0% (中間) | 50% (下位G)
```

- **上位G勝率**: 35.0%（中程度）
- **バランス**: 上下を同等に重視、中間を無視
- **長所**: シンプルな2値判定、実装が簡単
- **短所**: 中間領域の有用な情報を失却、複雑なパターンに対応困難
- **評価**: 平衡性は良いが、情報を過度に削減

### 3値バランス型 (45/10/45, 40/20/40, 36/28/36, 33/34/33)

```
L% (上位G) | M% (中間) | R% (下位G)  # L+M+R = 100%
```

**45/10/45**: 上位・下位重視、中間軽視
- 上位G勝率: **最高（38.3%）**
- 曜日別上位G: **最高クラス（36.5%）**
- **推奨**: 機種名予測が必要なケース

**40/20/40**: 均等寄り、中間領域活用
- 中間G勝率: **最高（41.7%）**
- 台末尾での強さ: 中間G勝利 60%
- **推奨**: 台末尾パターン分析が必要なケース

**36/28/36 (現在)**: 比較的均等な配分
- 全体的にバランス型
- 上位G: 33.3%（中位）
- 中間G: 38.3%（上位）
- 下位G: 28.3%（下位）
- **評価**: 万能型だが「最高」ではない

**33/34/33**: ほぼ均等分割
- 最も均等な配分
- 全体的に平凡な成績
- **評価**: 安定性は高いが精度は低い

### 中間重視型 (20/60/20, 10/80/10)

```
20% (上位G) | 60% (中間) | 20% (下位G)
10% (上位G) | 80% (中間) | 10% (下位G)
```

**20/60/20**: 中間領域への適度な重点
- 下位G勝率: 最高（41.7%）
- 安定した下位グループ検出
- **使用**: 負け組検出が重要なケース

**10/80/10**: 中間領域への極度な重点
- 下位G勝率: 第2位（36.7%）
- 機械番号での下位G勝率: 60%（最高）
- 曜日別の変動が大きい
- **評価**: 特定属性に特化、汎用性は低い

## 推奨される最適比率

### 全体パフォーマンス（DD別）スコアリング

各分割比率を統合評価（上位G重視、中間G、下位G のバランス）

```
スコア = (上位G勝率 × 1.2) + (中間G勝率 × 1.0) + (下位G勝率 × 0.8)
# 上位Gを重視（予測精度として最も重要）
```

| 分割比率 | スコア | ランク | 評価 |
|---------|--------|--------|------|
| **45/10/45** | 40.13 | **1位** | **推奨** |
| 40/20/40 | 37.34 | 2位 | 次点 |
| 36/28/36 | 36.60 | 3位 | 現在 |
| 50/0/50 | 35.53 | 4位 | 代替案 |
| 20/60/20 | 34.40 | 5位 | 専用 |
| 33/34/33 | 33.63 | 6位 | 非推奨 |
| 10/80/10 | 33.93 | 7位 | 実験用 |

### 最適比率: **45/10/45**

#### 推奨理由

1. **上位G勝率が最高（38.3%）**
   - 3属性均等での上位グループ検出精度が最も高い
   - 予測精度として重要な指標

2. **機種名での強い予測能力（45.0%）**
   - 最も信頼性の高い属性での勝率が高い
   - 実務的な利用価値が高い

3. **曜日別パフォーマンスが良好（36.5%）**
   - 日変動への耐性が高い
   - 広い運用条件での安定性

4. **中間領域への適切な考慮（36.7%）**
   - 中程度の学習対象を効率的に活用
   - 極端な削減（50/0/50）より情報損失が小さい

5. **台末尾での柔軟性（40.0%）**
   - パターン系属性での対応可能

## 現在の 36/28/36 との比較

### 改善ポイント

```
45/10/45 vs 36/28/36
─────────────────────────

上位G勝率: 38.3% → 33.3% (↑4.8 pp) ★最大改善
機種名:   45.0% → 45.0% (=)
台末尾:   40.0% → 35.0% (↓5.0 pp)

中間G勝率: 36.7% vs 38.3% (↓1.6 pp)
下位G勝率: 25.0% vs 28.3% (↓3.3 pp)
```

### 変更による影響

| 指標 | 現在 (36/28/36) | 提案 (45/10/45) | 変化 | 影響度 |
|------|---------|---------|--------|--------|
| 上位G精度 | 33.3% | 38.3% | +5.0pp | 大（重要） |
| 全体スコア | 36.60 | 40.13 | +3.53 | 中 |
| 機種名精度 | 45.0% | 45.0% | 同等 | 無 |
| 台末尾精度 | 35.0% | 40.0% | +5.0pp | 中 |
| バランス性 | 高 | 中 | -0.5 | 微 |

### 判定

**45/10/45 への変更は推奨**

- 上位G検出精度が +5.0pp の改善
- 実務的には「高精度の上位グループ検出」が最優先
- 中間G・下位G の若干の低下は許容範囲
- 40/20/40 も有力だが、上位G精度で劣る（26.7% vs 38.3%）
